Latest Updates

## Earthquake Hub Software — What’s New
A smoother, clearer, more informative earthquake map experience.

- ***Clearer maps with more layers:*** Switch between map styles and view fault lines and tectonic plates with tap-to-learn details.
- ***Live earthquake and station updates:*** Earthquakes and station status now update in real time, keeping information current.
- ***Better search and browsing:*** Improved filters, sorting, and presets make it easier to find events.
- ***Improved mobile experience:*** Cleaner controls, smarter panels, and easier navigation, especially in landscape mode.
- ***More readable popups and overlays:*** Map labels and information windows adjust automatically and stay out of the way.
- ***Clearer alerts and notifications:*** In-app messages now provide better context and feedback.
- ***Easier account recovery:*** Password reset and recovery flows are simpler and more user-friendly.
- ***Cleaner device status display:*** Station and device statuses are more consistent, readable, and easier to scan.
- ***Accessibility and performance upgrades:*** Faster map loading, smoother interactions, and improved keyboard support.
- ***Privacy-respecting analytics:*** Usage tracking now follows consent-compliant standards.

## Sender Client Software — What’s New
More reliable monitoring, clearer health checks, and safer recovery tools for station owners.

- ***New device health checks:*** Network status, time synchronization, and system resource health are now visible.
- ***Improved device linking and recovery:*** Linking is more reliable, with better handling when issues occur.
- ***Cleaner device removal:*** Unlinking a device now fully removes related data to prevent future problems.
- ***Better stream monitoring:*** Shows when data was last successfully sent, helping detect issues earlier.
- ***Clearer monitoring dashboard:*** Setup and monitoring are separated into clearer views.
- ***Refined interface and visuals:*** Improved layout, tooltips, and dialogs across the app.
- ***Built-in recovery tools:*** Devices can be reset or recovered without manual troubleshooting.
